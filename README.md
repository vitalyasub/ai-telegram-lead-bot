# Telegram Lead Bot (AI Automation Example)

Телеграм-бот для збору заявок від клієнтів з автоматичним записом у Google Sheets та повідомленням адміністратора.

Проект демонструє базову автоматизацію бізнес-процесів за допомогою Telegram Bot API та Python.

---

# 🚀 Можливості

- 📩 Збір заявки від користувача через Telegram
- 📊 Автоматичний запис заявки у Google Sheets
- 🔔 Повідомлення адміністратора про нову заявку
- 📄 Перегляд останніх заявок через адмін-панель
- 🔎 Пошук заявки за номером телефону
- 🧠 FSM (Finite State Machine) для коректного збору даних

---

# 🧱 Стек технологій

- Python
- aiogram
- Google Sheets API
- gspread
- python-dotenv
